目录

  4. 部署github，生成免费博客网站

一、下载安装hugo

Mac安装方式

  • brew install hugo
  • hugo version 查看hugo版本

Window安装方式

  • 去Hugo releases 页面下载hugo_xxx_Windows-64bit.zip

  • 解压，找个目录解压出hugo.exe

  • 将路径写进系统变量里(path)

  • 启动控制台 输入 hugo version
